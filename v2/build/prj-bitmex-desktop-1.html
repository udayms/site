CLI Trading

Command Line Interface for Trading

During the 3 years I was with BitMEX, I along with the UX team initiated and ran multiple blue sky ideas with multiple goals like increasing user engagement on the platform, making trading safer for newbie traders and also making trading quicker for expert traders. The CLI trading feature was one such experiment that geneated a lot of excitement within the organziation leadership team.

The core idea behind this exploration was to create a command line interface that expert traders can use to execute trades using a combination of simple commands. For ex: +1.2k @46235 /5x, will place an order for buying 1200 contracts (of the instrument that user is on) at a limit price of $46235 using a leverage of 5x.

Prototype Goals
  • Create an immersive and interactive prototype on the testnet environment
  • Show preview of the order as the user types the command
  • Support simple commands like navigation, simple buy/sell orders & more complex commands to execute multiple orders in a single line using market data
Prototyping Stack
  • Tampermonkey, Chrome DevTools
  • NodeJS, Postman API tool, VS Code debugger
Design Wins
  • High excitement in early user tests and SME feedback sessions
  • Retail traders saw this as the closest they can get to feel like a quant trader
  • Task completion sped up by 46% in controlled test sessions
  • Seeing the prototype working within testnet environment helped raise confidence in engineering counterparts on the feasibility of the idea
